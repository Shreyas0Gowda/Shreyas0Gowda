## 🌟 Featured Projects

### ⚖️ [LEXA – Autonomous Multi-Agent Courtroom Intelligence System](https://github.com/Shreyas0Gowda/courtroomsimu)
*An advanced legal technology simulation engine that orchestrates concurrent AI agents to analyze evidence and mock trials.*
- **Technical Stack:** Python, FastAPI, React, TypeScript, LangGraph, NVIDIA NIM APIs, FAISS Vector DB, Supabase.
- **Key Achievements & Workflow:**
  - Engineered an **8-phase asynchronous workflow graph** mapping nodes for Specialized Agents: *Evidence Extractor, Legal Researcher, Prosecution, Defense, Contradiction Detector, Judge, Jury, and Appeal Court*.
  - Configured high-performance vector retrieval utilizing **FAISS indexing** to query relevant Indian Code regulations.
  - Implemented standard containerized deployment paths with **Docker** and secure authentication callbacks using **Supabase**.
- **Links:** [Source Code](https://github.com/Shreyas0Gowda/courtroomsimu) | [System Schema](https://github.com/Shreyas0Gowda/courtroomsimu#architecture)
